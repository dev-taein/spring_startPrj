# Spring  댓글 게시판 프로젝트 (개인 프로젝트)
## 목적 ##
+ 웹 개발자의 필수요소 Spring
+ Spring프레임워크 사용 경험을 극대화
+ Spring프레임워크의 기술 습득
+ 웹 개발자로써 한 단계 더 성장하기 위한 프레임워크에 대한 이해와 기술 연마

# 개발 환경 및 사용한 언어
> Front-End
+ Bootstrap v3.2.0
+ jQuery v1.10.2
+ CSS 3
+ HTML 5
> Back-End
+ Spring v4.3.1.RELEASE
+ Maven v2.9
+ MyBatis v3.2.8
+ JDK v1.8
+ Tomcat v9.0
+ MySQL v8.0
